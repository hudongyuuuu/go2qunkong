# GO2 控制面板 v4.0 - 舞蹈增强版

## 🎉 主要更新

### 1. ⚠️ 修复运动指令问题
- **修复**：前进、后退、左移、右移、左转、右转指令
- **原因**：之前使用的 Move 指令参数格式不正确
- **现在**：使用正确的 x, y, z 参数格式

### 2. 🎭 完整的 GO2 Air 动作库
添加 **30+ 个内置动作**，分为 6 大类：

- **基础动作**（4个）：站立、蹲下、打招呼、停止
- **运动动作**（6个）：前进、后退、左移、右移、左转、右转
- **舞蹈动作**（5个）：舞蹈1-5，时长 8-12 秒
- **特技动作**（6个）：跳跃、伸展、坐下、倒立、翻滚、空翻
- **交互动作**（4个）：乞食、握手、击掌、和平手势
- **步态动作**（4个）：小跑、踱步、跳跃跑、飞奔

### 3. 🎼 动作序列编辑器
- 可视化编辑舞蹈序列
- 自定义每个动作的时长
- 保存/加载序列（JSON 格式）
- 实时显示当前执行进度
- 支持随时中断

### 4. 🎵 预设 3 分钟舞蹈
- **60+ 个精心编排的动作**
- **5 个主题段落**：开场、快节奏、互动、技巧、节奏
- **统计**：6 大类动作均衡分布
- **时长**：180 秒（3 分钟）
